# Strikketeller

A very basic knitting counter for tracking rows and repetitions.

## Building the project from source

### Prerequisites

- Node.js 20 or higher
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Deployment

The application is automatically deployed to GitHub Pages on pushes to the `main` branch.
